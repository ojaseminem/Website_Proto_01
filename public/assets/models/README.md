# Sea Turtle Model Source

Preferred production GLB path:

```text
/assets/models/sea-turtle.glb
```

Selected source:

- Model: "Sea Turtle"
- Creator: Ignas / Ignas3D
- Source: https://sketchfab.com/3d-models/sea-turtle-ddb6789834f0412090863e8fd3c0c4db

Sketchfab download URLs are not stable hotlink targets, so the website code loads
a site-owned GLB from this folder and falls back to the built-in procedural sea
turtle if the file is unavailable. Check and preserve the model's current
license terms from Sketchfab before deploying a downloaded copy.
